# Web Inventory Sistem - Jaya Motor Sbahar

Sistem informasi manajemen inventaris sparepart motor berbasis web.

## Teknologi
- React.js + Vite
- Tailwind CSS
- LocalStorage (sementara)

## Fitur
- Login & Autentikasi
- Dashboard + Low Stock Alert
- Tambah Barang Masuk
- Penjualan & Keluar Barang
- Kelola Data Barang (CRUD)
- Koreksi Stok + Audit Trail
- Laporan
